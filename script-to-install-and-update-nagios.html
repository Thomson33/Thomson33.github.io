Script to install and update Nagios

Posted on Fri 30 November 2018 in DevOps

Files and last updates : https://github.com/thoomson/InstallUpdateNagios

Python script to install Nagios on Debian 9

Prepare your setup

To use this script you need to install Python3 and pip3. * On debian/ubuntu : apt install python3 * On centos : yum install python3 * On Windows : install the last 3.X version on : https://www.python.org/downloads/

In the installation, be carrefull to select the option Add Python to environnment variables

When that was done, you have to edit the script variables.py with your own informations.

Run the script

python3 install.py

python3 update.py
